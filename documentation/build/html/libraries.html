List of pyacs core librariesÂ¶

  • pyacs.lib.astrotime module: core functions for dates

  • pyacs.lib.coordinates module: core functions for coordinates

  • pyacs.lib.euler module: core functions for Euler poles

  • pyacs.lib.faultslip module: simple core functions for fault slip

  • pyacs.lib.glinalg module: core functions for usual linear algebra operations in geodetic problems

  • pyacs.lib.gpstime module: functions for gps dates

  • pyacs.lib.robustestimators module: a set of linear robust estimators

  • pyacs.lib.shapefile module: functions for shapefiles conversions. Useful to visualize results with QGis

  • pyacs.lib.strain module: strain rate linear estimation

  • pyacs.lib.units module: a few unit conversions functions

  • pyacs.lib.utils module: some useful functions for pyacs developpers

  • pyacs.lib.vel_field module: horizontal velocity field analysis (Euler pole, profiles, strain rate)
